# dsa-practice

Personal practice repository for data structures and algorithms problems.

## Structure

- `leetcode/` — problem solutions, one file per problem
- `patterns/` — pattern guides with explanations and examples
- `schedule.md` — spaced repetition review tracker

## Method

Problems are organized by pattern. After solving a problem, it's logged in `schedule.md` with a mastery score and a calculated next-review date. Review intervals increase as mastery improves (1d → 2d → 4d → 7d → 14d).
