# Bitwise Operations Explorer
**Topics:** Bitwise operators, Output operations, Selection structures

**Problem Description:**
Write a program that performs various bitwise operations on two integers and displays the results.

**Input:**
- Two integers a and b

**Output:**
- a AND b
- a OR b
- a XOR b
- NOT a (bitwise complement)
- a left shift by 1
- b right shift by 1

**Sample Input:**
```
12
5
```

**Sample Output:**
```
AND: 4
OR: 13
XOR: 9
NOT a: -13
Left Shift a: 24
Right Shift b: 2
```
