# Subsequence Extrema

### Description
A subsequence is a sequence that can be derived from another sequence by deleting some or no elements without changing the order of the remaining elements. Given a string S and an integer K, your task is to find two specific subsequences of length K: the one that comes first lexicographically (the smallest) and the one that comes last lexicographically (the largest).

Lexicographical comparison means comparing strings character by character, like in a dictionary. For example, "apple" is lexicographically smaller than "apricot" because at the third character, 'p' comes before 'r'.

### Input
- The first line of input will be a string S containing only lowercase English letters (1 ≤ length of S ≤ 100)
- The second line of input will be an integer K (1 ≤ K ≤ length of S)
