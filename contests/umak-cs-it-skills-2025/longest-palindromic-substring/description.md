# Longest Palindromic Substring

### Description
Given a string, find the longest substring that is a palindrome (reads the same forwards and backwards). If there are multiple palindromic substrings of the same maximum length, output the first one found (leftmost).

Additionally, count the total number of distinct palindromic substrings in the string.

### Input
- A single string S (1 ≤ length ≤ 1000, only lowercase letters)

### Output
- First line: The longest palindromic substring
- Second line: "Length: X"
- Third line: "Total palindromic substrings: Y"

### Sample Test Cases

#### Sample 1
```
Input:
babad

Output:
bab
Length: 3
Total palindromic substrings: 7
```

#### Sample 2
```
Input:
cbbd

Output:
bb
Length: 2
Total palindromic substrings: 5
```

#### Sample 3
```
Input:
racecar

Output:
racecar
Length: 7
Total palindromic substrings: 10
```

#### Sample 4
```
Input:
abcdef

Output:
a
Length: 1
Total palindromic substrings: 6
```
